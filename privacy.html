Privacy Policy — Easy Read PDF

Last updated: May 13, 2026

Overview

Easy Read PDF is a privacy-respecting PDF reader app designed to help users read PDF files in larger, easier-to-read text. This Privacy Policy explains what data we collect (we collect very little) and how we handle it.

Data We Do NOT Collect

Easy Read PDF does not:

  • Send your PDF files to any server
  • Track your usage or behavior
  • Use analytics services
  • Display advertisements
  • Require user registration or accounts
  • Collect personal information
  • Share data with third parties

How PDF Files Are Processed

All PDF processing happens entirely on your device. When you select a PDF file, the app reads it locally in your browser/device memory to extract text and display it. The PDF file is never uploaded to any server.

Local Storage

The app saves the following preferences in your device's local storage (these never leave your device):

  • Font size preference
  • Theme preference (light/dark)
  • iOS install prompt dismissal state

Permissions

The app requests file access permission only to read PDF files you explicitly select. No background file access occurs.

Third-Party Services

The app loads web fonts from Google Fonts (fonts.googleapis.com, fonts.gstatic.com) to display Korean and English text. Google may receive standard web request information when these fonts are loaded. After the first load, fonts are cached locally and can be used offline.

Children's Privacy

The app is suitable for all ages and does not knowingly collect data from children under 13.

Changes to This Policy

Any updates to this Privacy Policy will be posted on this page with an updated revision date.

Contact

For questions about this Privacy Policy, please open an issue at the app's GitHub repository: https://github.com/apps-and-downs/easy-read-pdf

개인정보 처리방침 — Easy Read PDF

최종 업데이트: 2026년 5월 13일

개요

Easy Read PDF는 PDF 파일을 큰 글자로 편하게 읽을 수 있도록 도와주는 프라이버시 중심 PDF 리더 앱입니다. 본 방침은 본 앱이 수집하는 데이터(매우 적습니다)와 그 처리 방식을 설명합니다.

수집하지 않는 정보

본 앱은 다음을 하지 않습니다:

  • PDF 파일을 외부 서버로 전송하지 않습니다
  • 사용 행동을 추적하지 않습니다
  • 분석 도구를 사용하지 않습니다
  • 광고를 표시하지 않습니다
  • 회원 가입이나 계정을 요구하지 않습니다
  • 개인정보를 수집하지 않습니다
  • 제3자에게 데이터를 공유하지 않습니다

PDF 파일 처리 방식

모든 PDF 처리는 사용자의 기기 내에서만 이루어집니다. 사용자가 PDF 파일을 선택하면, 앱은 브라우저/기기 메모리 안에서만 해당 파일을 읽어 텍스트를 추출하고 표시합니다. PDF 파일은 어떤 서버에도 업로드되지 않습니다.

로컬 저장

다음 환경설정만 기기의 로컬 저장소에 보관됩니다 (기기 밖으로 나가지 않습니다):

  • 글자 크기 설정
  • 테마 설정 (라이트/다크)
  • iOS 설치 안내 닫힘 상태

권한

앱은 사용자가 명시적으로 선택한 PDF 파일을 읽기 위해서만 파일 접근 권한을 요청합니다. 백그라운드에서의 파일 접근은 발생하지 않습니다.

제3자 서비스

본 앱은 한국어와 영어를 표시하기 위해 Google Fonts(fonts.googleapis.com, fonts.gstatic.com)에서 웹 폰트를 불러옵니다. 폰트가 로드될 때 표준 웹 요청 정보가 Google에 전달될 수 있습니다. 첫 로드 후에는 폰트가 로컬에 캐시되어 오프라인에서도 사용 가능합니다.

아동의 개인정보

본 앱은 모든 연령대에 적합하며, 13세 미만 아동의 정보를 의도적으로 수집하지 않습니다.

방침의 변경

본 개인정보 처리방침의 변경 사항은 본 페이지에 업데이트 날짜와 함께 게시됩니다.
